# fluttershare

> 1). Create firebase project
>
> 2). Follow steps to create project
>
> 3). Link firebase with flutter by adding googleservices.json file in project
>
> 4). Run following commands in terminal
>```
> $ flutter pub get
> 
> $ flutter pub upgrade
> 
> $ flutter run
> ```
